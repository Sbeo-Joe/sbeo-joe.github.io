OpenAI가 야심 차게 출시했던 AI 동영상 생성 앱 <mark>Sora</mark> 를 전격 폐기했다. 동시에 코딩 에이전트 <mark>Codex</mark> 에는 플러그인 기능을 추가해 Claude Code와의 격차를 좁혔다. 단 하루 만에 이뤄진 이 두 결정은 OpenAI의 AI 제품 전략이 근본적으로 재편되고 있음을 보여준다.

## 핵심 포인트

- OpenAI, 비디오 생성 앱 Sora 전면 폐기 + ChatGPT 내 영상 생성 계획 철회 + <mark>$10억 Disney 계약</mark> 청산
- Codex에 플러그인 기능 추가 — Ars Technica: "Claude Code와의 일부 격차를 좁혔다"
- <span class="hl-red">Sora 팀 고위 인사 역할 조정</span> — AI 비디오 생성 시장에서 OpenAI의 전략적 후퇴 공식화

## Sora: 왜 만들었고 왜 죽였나

Sora는 2024년 2월 공개 당시 업계를 충격에 빠뜨렸다. 텍스트 한 줄로 고화질 영상을 생성하는 능력이 영화·광고 산업의 판도를 바꿀 것이라는 기대가 쏟아졌다. <span class="hl-blue">$10억 규모의 Disney 파트너십</span> 이 체결된 것도 그 기대를 반영한 결과였다.

그러나 The Verge의 심층 보도에 따르면, OpenAI 내부에서 상황이 반전됐다. 화요일 아침까지 모든 것이 정상이었지만 하루 만에:

- 비디오 생성 앱 Sora 전면 폐기
- ChatGPT 내 영상 생성 통합 계획 백지화
- $10억 Disney 계약 청산
- 담당 임원 역할 대폭 축소

무엇이 이 결정을 촉발했는가? 복수의 요인이 겹쳤다고 분석된다.

| 요인 | 내용 |
|------|------|
| 경쟁 환경 악화 | Runway, Pika, Kling, Sora 대비 기술 우위 불명확 |
| 수익화 난항 | 비디오 생성의 컴퓨팅 비용 대비 수익 모델 미완 |
| 전략 집중 | ChatGPT·API·에이전트 수익원에 자원 재배분 필요 |
| 내부 리더십 | 담당 임원의 역할 변경으로 제품 방향성 상실 |

> "OpenAI는 지금 선택과 집중을 하고 있다. Sora에 쏟던 자원을 ChatGPT와 에이전트 인프라로 이동시키는 결정이다."

## Codex 플러그인: Claude Code를 추격하는 신호

Sora 폐기와 같은 날, OpenAI는 코딩 에이전트 Codex에 플러그인 기능을 공식 추가했다. Ars Technica는 "OpenAI가 Codex를 코딩을 넘어서는 영역으로 공식 확장했다"고 평가했다.

플러그인 도입의 의미는 크게 두 가지다.

**첫째, Claude Code와의 격차 축소.** <span class="hl-blue">Anthropic의 Claude Code는 이미 플러그인과 유사한 외부 도구 연동 기능을 제공</span> 해왔다. Codex의 플러그인 추가로 기능 면에서 일정 부분 동등해졌다.

**둘째, 코딩 에이전트에서 범용 작업 에이전트로의 전환.** 플러그인은 Codex가 단순 코드 작성을 넘어 외부 데이터베이스, API, 도구와 연동하는 것을 가능하게 한다. <span class="hl-blue">OpenAI의 에이전트 전략이 구체화되는 중</span> 이다.

## AI 제품 전쟁의 새 구도

OpenAI의 이날 결정은 AI 제품 전략의 우선순위를 분명히 한다. 비디오 생성은 포기하고, 코딩·에이전트 영역을 강화한다. 이는 Anthropic(Claude Code), Google(Gemini 코딩), GitHub Copilot과의 직접 경쟁을 선택하겠다는 선언이다.

<span class="hl-red">리스크는 분명하다.</span> Sora에 투자한 Disney와의 신뢰 관계, 비디오 AI 시장에서의 선점 기회를 모두 잃었다. Runway, Pika 등 후발 주자들이 그 공백을 채울 가능성이 높다.

<span class="hl-blue">반면 기회는 더 크다.</span> 코딩 에이전트와 범용 AI 에이전트 시장의 규모는 비디오 생성 시장을 압도한다. 기업 고객들이 가장 먼저 지갑을 여는 영역이기도 하다. OpenAI가 Sora를 버리고 Codex를 키우는 결정은 이 계산에 근거한다.
